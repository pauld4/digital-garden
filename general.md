### **What is the difference between a Software Engineer and Software Developer?**

Both titles are similar, but engineers focus more on the overall functionality of an application, and developers focus more on specific functionalities of an application.

**Source:** https://careerengagement.tulane.edu/blog/2023/08/05/software-developer-vs-software-engineer-whats-the-difference/

#### **What is Quality Assurance(QA)?**

The definition of "assurance" is "a positive declaration intended to give confidence". In software testing, quality assurance simply assures that the software application works as it should. Quality assurance is done in parallel to developing the software, so the cycle goes back and forth between implementing a feature in the application and testing that feature.

**Source:** https://www.geeksforgeeks.org/software-testing/software-engineering-software-quality-assurance/

#### **What is JavaScipt(JS)?**

JavaScript works with HTML and CSS on a website's page and its data to create interactive and dynamic webpages. JavaScript allows a webpage to detect when a user clicks, presses a keyboard key, or other interactions, and can dynamically change a web page based on which interaction the user performs.

**Source:** https://www.w3schools.com/whatis/whatis_js.asp
**Docs:** https://developer.mozilla.org/en-US/docs/Web/JavaScript

#### **What is PHP(PHP Hypertext Preprocessor)?**

PHP is a language read by a website's server and usually handles user information or connecting to a database. Any time a user submits a form, or logs into a website, PHP is used to verify and authenticate users and information submitted by the user.

**Docs:** https://www.php.net/docs.php

#### **What is HTML(Hypertext Markup Language)?**

HTML is the language that structures the basic layout of a website and what elements are displayed.

**Docs:** https://developer.mozilla.org/en-US/docs/Web/HTML
**Docs:** https://www.w3schools.com/html/

#### **What is CSS(Cascading Style Sheets)?**

CSS is used to style the HTML elements drawn on a website. A button in HTML will look the same across every website, but CSS is used to style that button based on what the developer of the website wants.

**Docs:** https://developer.mozilla.org/en-US/docs/Web/CSS
**Docs:** https://www.w3schools.com/css/

#### **What is the difference between Front End and Back End?**

Front end is the part that the user sees when visiting a website, such as HTML, CSS, and JavaScript, and back end is the server side of a website that handles user input and information, using PHP. A website will look good because the front end is designed well, and a website will function well because the back end is well developed.

**Source:** https://www.geeksforgeeks.org/blogs/frontend-vs-backend/
**Source:** https://aws.amazon.com/compare/the-difference-between-frontend-and-backend/

#### **What is user authentication?**

The step by step process that a user goes through when interacting with a website, such as logging in. In a password based authentication system, the website ensures that the username and password entered follow correct formatting, and then are sent over a secure connection to the server where it is checked against the corresponding username and hashed password in the database. If that information is correct, then the user is assigned a token which is sent back to the user's client.

**Source:** https://www.microsoft.com/en-us/security/business/security-101/what-is-authentication

#### **What are database driven features?**

Features of a website that use specific information from the database to provide dynamic content for the user. Sites such as Netflix use information that users submit, such as watching a show/movie multiple times and rating it, and use that information to provide suggestions to that user or other users who consume similar content.

**Source:** https://iacis.org/iis/2006/Kovacs.pdf
